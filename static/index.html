Timestamp Microservice

This is the page at the base url for the timestamp microservice. Usage is as follows:

  • Pass a unix time value: http://www.fancy-bison.glitch.me/14501376000
  • Pass a natural time value (date): http://fancy-bison.glitch.me/December%2015%202015

Either of these options will return the JSON {"unix":14501376000, "natural":"December 15, 2015"}
